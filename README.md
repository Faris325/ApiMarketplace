# 🚀 Marketplace API (Django + DRF + PostgreSQL + Docker)

Backend API для маркетплейса с товарами, пользователями, комментариями и историей цен.

---

# 🧱 Технологии

- Python 3.12
- Django 6
- Django REST Framework
- PostgreSQL 15
- Docker & Docker Compose
- SimpleJWT
- Djoser
- django-filter
- psycopg2
